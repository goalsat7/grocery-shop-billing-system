Grocery Billing System
Description

A Python GUI application for managing grocery store billing.
Users can select products and items, enter quantities, apply discounts and GST, generate a final bill, save it as Text or PDF, and print it.
Features
*Add, modify, and delete items in the bill
*Automatic total calculation
*Apply discount and GST
*Generate final bill
*Save as text file or PDF
*Print the bill
Requirements
*Python 3.x
*Tkinter (comes with Python)
*ReportLab (pip install reportlab)
How to Run
Install the required packages:
cmd
pip install reportlab
Run the program:
python grocery_billing_system.py
